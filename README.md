# Jira 2 Notion DB

## Installation

```shell
pip install jira-2-notion-db
```

## Configuration

* Jira login token
* Notion integration token


## Usage

* arguments:
  * --url URL
    * Jira url ex) https://your.atlassian.net
  * --username
    * Jira username
  * --password
    * Jira api token (Not Password)
  * --project
    * Jira project name
  * --notion
    * Notion integration bot token
  * --delay
    * Notion update delay
